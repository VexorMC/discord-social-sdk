# Discord Social SDK
The Discord Social SDK is a powerful tool that allows developers to integrate Discord's social features into their applications. With this SDK, you can easily add functionalities such as user authentication, friend lists, messaging, and more.

This repository contains the Java bindings for the Discord Social SDK, enabling Java developers to leverage Discord's social capabilities in their applications.

## Contributing
We welcome contributions from the community! If you would like to contribute to these bindings, please follow these steps:
1. Fork the repository.
2. Make your changes and ensure that your code adheres to the project's coding standards.
3. Submit a pull request with a clear description of your changes.
4. We will review your pull request and provide feedback as necessary.
5. Once approved, your changes will be merged into the main branch.
